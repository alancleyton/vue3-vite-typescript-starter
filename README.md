Start a new Vue 3 project with Vite, Typescript, Jest, ESlint and Prettier.
## Installation

Using `npm` install all dependencies.
```bash
  npm install
```

## Development
Run the project on development mode.
```bash
  npm run dev
```

lint and fix all files.
```bash
  npm run lint
```

## Build for production
```bash
  npm run build
```
